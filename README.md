A collection of my Dotfiles for the configuration of:
- Terminal (Ghostty)
    - Keybindings
    - Shaders
    - Split settings
    - Border Handling
- Neovim
    - Keybingings/Shortcuts
    - Dashboard
    - QOL improvements
- Yatoro (Music Player linked to Apple Music)
    - Theme
    - Album Cover Settings
- Bash Profile
    - Aliases
    - PS1 and PS2 (prompt) customization
